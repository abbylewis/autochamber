# Data from GENX and Chapada flux chambers

Formatted data from GENX

## Usage

``` r
data_small_genx
```

## Format

### `data_small_genx`

A data frame with 73,800 rows and 7 columns:

- TIMESTAMP:

  Timestamp

- CH4d_ppm:

  Methane concentration (ppm)

- CO2d_ppm:

  Carbon dioxide concentration (ppm)

- N2Od_ppm:

  Nitrous oxide concentration (ppm)

- MIU_VALVE:

  Sampling valve (chamber)

- Manifold_Timer:

  Time since switching to this valve

- Format:

  Data format (only NEW works for this processing code)
