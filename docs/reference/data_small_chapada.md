# Formatted data from Chapada

Formatted data from Chapada

## Usage

``` r
data_small_chapada
```

## Format

### `data_small_chapada`

A data frame with 80,130 rows and 7 columns:

- location:

  Site name

- TIMESTAMP:

  Timestamp

- CH4d_ppm:

  Methane concentration (ppm)

- CO2d_ppm:

  Carbon dioxide concentration (ppm)

- Chamber:

  Sampling chamber

- Manifold_Timer:

  Time since switching to this valve

- Flux_Status:

  Status of chamber (e.g., open/closed)
